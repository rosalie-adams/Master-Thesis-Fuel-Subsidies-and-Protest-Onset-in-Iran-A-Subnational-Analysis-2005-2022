# Master-Thesis-Fuel-Subsidies-and-Protest-Onset-in-Iran-A-Subnational-Analysis-2005-2022
Replication materials for 'Fuel Subsidies and Protest Onset in Iran: A Subnational Analysis, 2005–2022' (Adams, 2026). Includes data construction scripts (Python/R), 2SLS analysis with robustness checks, and panel data covering 30 Iranian provinces, 2005–2022.
